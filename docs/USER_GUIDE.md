## 1. 이게 뭔가요?

**지오영, 복산, 인천, 티제이팜, 백제, HMP, 피코, 새로팜, 신덕팜, 대전동원약품** 10개 도매상의 재고를 **한 번에 검색**하고, 가장 싼 곳에서 바로 주문할 수 있는 프로그램입니다.

### 할 수 있는 것

- 제품명이나 보험코드로 8개 도매 동시 검색
- 검색 결과에서 바로 주문 (로그인 자동)
- 품절약이 입고되면 자동으로 주문 (긴급주문)
- 가격이 바뀌면 텔레그램으로 알림
- Claude AI에게 말로 검색/주문 시키기

### 비용

- **무료**입니다 (로컬 설치형)
- 클라우드 서비스(24시간 자동 모니터링)는 팜스퀘어 유료회원 대상

## 2. 설치하기

### 방법 A: 간편 설치 (권장)

#### Windows

1. **다운로드**
   - 아래 링크에서 최신 버전 다운로드:
   - https://github.com/maipharm/maipharm-domae-mcp/releases/latest
   - `maipharm-domae-mcp-v1.0.0.zip` 파일 다운로드
   - 원하는 폴더에 압축 해제 (예: `C:\maipharm-domae-mcp`)

2. **설치**
   - 압축 해제한 폴더에서 `install.bat` 더블클릭
   - Python이 없으면 **자동으로 설치**합니다 (인터넷 연결 필요)
   - "설치 완료" 메시지가 나오면 끝

3. **실행**
   - `start.bat` 더블클릭
   - 브라우저가 자동으로 열립니다 (http://localhost:5900)

> Python을 따로 설치할 필요가 없습니다. `install.bat`이 알아서 해줍니다.

#### Mac

1. 터미널 열기 (Spotlight에서 "터미널" 검색)
2. 아래 명령어 복사해서 붙여넣기:

```bash
curl -fsSL https://github.com/maipharm/maipharm-domae-mcp/releases/latest/download/install.sh | bash
```

3. 설치 완료 후:

```bash
python3 -m domae_mcp
```

4. 브라우저에서 http://localhost:5900 접속

### 방법 B: 수동 설치 (개발자용)

```bash
git clone https://github.com/maipharm/maipharm-domae-mcp.git
cd maipharm-domae-mcp
pip install -e .
python -m domae_mcp
```

## 3. 처음 설정하기

프로그램을 처음 실행하면 설정이 필요합니다.

### 3-1. API 키 등록

1. 상단 메뉴에서 **[설정]** 클릭
2. **API 키** 탭 선택
3. 팜스퀘어(pharmsquare.com)에서 무료 회원가입
4. 마이페이지 → "도매 API 키 발급" → 키 복사
5. API 키 입력란에 붙여넣기 → **[검증]** 클릭

### 3-2. 도매 계정 등록

1. **[설정]** → **도매 계정** 탭
2. 사용 중인 도매상의 아이디/비밀번호 입력
3. **[테스트]** 버튼으로 로그인 확인
4. 녹색 "로그인 성공" → 정상 등록

```
예시:
┌──────────┬──────────────┬──────────┬────────┐
│ 도매상    │ 아이디        │ 비밀번호  │ 상태   │
├──────────┼──────────────┼──────────┼────────┤
│ 지오영    │ myph01       │ ****     │ ✅ 성공 │
│ 복산     │ boksan123    │ ****     │ ✅ 성공 │
│ 인천     │              │          │ 미설정  │
│ ...      │              │          │        │
└──────────┴──────────────┴──────────┴────────┘
```

> 비밀번호는 암호화되어 내 PC에만 저장됩니다. 외부로 전송되지 않습니다.

> 모든 도매를 다 등록할 필요 없습니다. 거래하는 도매만 등록하세요.

## 4. 의약품 검색하기

1. 상단 메뉴에서 **[통합검색]** 클릭
2. 검색창에 제품명 또는 보험코드 입력

```
예: "아모잘탄" 또는 "655801440"
```

3. **[검색]** 클릭 (또는 Enter)
4. 모든 도매 결과가 가격순으로 정렬되어 표시

```
검색 결과 예시:
┌──────────────────┬──────┬────────┬────────┬──────┬────────┐
│ 제품명            │ 단위 │ 도매    │ 재고   │ 가격  │        │
├──────────────────┼──────┼────────┼────────┼──────┼────────┤
│ 아모잘탄정 5/100  │ 30T  │ 복산   │ 10     │ 12,300│ [주문] │
│ 아모잘탄정 5/100  │ 30T  │ 지오영  │ 25     │ 12,500│ [주문] │
│ 아모잘탄정 5/100  │ 30T  │ 백제   │ 5      │ 12,800│ [주문] │
└──────────────────┴──────┴────────┴────────┴──────┴────────┘
```

> 계정을 등록한 도매만 검색됩니다.

## 5. 주문하기

1. 검색 결과에서 원하는 도매의 행을 찾기
2. 수량 입력 (기본 1개)
3. **[주문]** 버튼 클릭
4. "주문 완료" 메시지 확인

> 주문이 되면 해당 도매 사이트에서도 주문 내역이 확인됩니다.

### 주문 이력 확인

- 상단 메뉴 **[주문이력]** 에서 과거 주문 내역을 확인할 수 있습니다.

## 6. 긴급주문 (품절약 자동 확보)

품절인 의약품이 어딘가에 재고가 들어오면 **자동으로 주문**하는 기능입니다.

### 등록 방법

1. 상단 메뉴 **[긴급주문]** 클릭
2. 제품 정보 입력:
   - 제품명: 아모잘탄정 5/100mg
   - 단위: 30T
   - 필요 수량: 10
3. 주문할 도매상 추가:
   - 도매상: 지오영, 제품코드: GEO-12345, 가격: 12500
   - 도매상: 복산, 제품코드: BOK-67890, 가격: 12300
4. **[등록]** 클릭

> 제품코드는 통합검색 결과에서 확인할 수 있습니다.

### 동작 원리

```
모니터링 실행 중일 때:
  → 60분마다 등록된 제품 자동 검색
  → 재고 발견!
  → 자동으로 주문 실행
  → 텔레그램으로 알림 전송
  → 필요 수량 채워지면 자동 종료
```

## 7. 가격 모니터링

관심 있는 의약품의 가격/재고 변동을 추적합니다.

### 설정 방법

1. 상단 메뉴 **[모니터링]** 클릭
2. 감시할 제품 추가:
   - 보험코드(예: 655801440) 또는 제품명 입력
   - **[추가]** 클릭
3. **[모니터링 시작]** 클릭

### 알림을 받으려면

- 텔레그램 설정이 되어 있어야 합니다 (아래 8번 참고)
- 가격이 바뀌거나, 재고가 새로 생기면 텔레그램으로 알림

### 주의사항

- 모니터링은 **PC가 켜져있고 프로그램이 실행 중일 때만** 동작합니다
- 모니터링 간격은 최소 60분입니다
- 24시간 자동 모니터링은 [클라우드 서비스](#13-클라우드-서비스-유료)를 이용하세요

## 8. 텔레그램 알림 설정

가격 변동, 긴급주문 체결 등의 알림을 텔레그램으로 받을 수 있습니다.

### 텔레그램 봇 만들기 (최초 1회)

1. 텔레그램 앱에서 **@BotFather** 검색 → 대화 시작
2. `/newbot` 입력
3. 봇 이름 입력 (예: 마이약국 도매알림)
4. 봇 사용자명 입력 (예: my_pharmacy_bot)
5. **봇 토큰**이 발급됩니다 → 복사해두기

```
예: 7123456789:AAHxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
```

### Chat ID 알아내기

1. 방금 만든 봇에게 아무 메시지 보내기 (예: "안녕")
2. 브라우저에서 아래 주소 접속 (토큰 부분을 본인 것으로 교체):

```
https://api.telegram.org/bot7123456789:AAHxxx.../getUpdates
```

3. 응답에서 `"chat":{"id":숫자}` 부분의 숫자가 Chat ID

### 프로그램에 등록

1. **[설정]** → **텔레그램** 탭
2. 봇 토큰 입력
3. Chat ID 입력
4. **[저장]** 클릭
5. **[테스트 발송]** 으로 확인 → 텔레그램에 메시지가 오면 성공

## 9. Claude AI로 사용하기

Claude Desktop 앱이 있으면, AI에게 말로 검색/주문을 시킬 수 있습니다.

### 설정 방법

1. Claude Desktop 설정 파일을 엽니다:
   - Windows: `%APPDATA%\Claude\claude_desktop_config.json`
   - Mac: `~/Library/Application Support/Claude/claude_desktop_config.json`

2. 아래 내용을 추가합니다:

```json
{
  "mcpServers": {
    "domae": {
      "command": "python",
      "args": ["-m", "domae_mcp", "--mcp"]
    }
  }
}
```

3. Claude Desktop을 재시작합니다

### 사용 예시

Claude에게 이렇게 말하면 됩니다:

- "아모잘탄 검색해줘"
- "지오영에서 아모잘탄 5개 주문해줘"
- "모니터링에 655801440 추가해줘"
- "긴급주문 목록 보여줘"
- "모니터링 시작해줘"

## 10. PC 켤 때 자동 실행

Windows에서 PC를 켤 때 자동으로 실행되도록 설정할 수 있습니다.

### 등록

```
start.bat를 더블클릭한 상태에서:
설정 → 시스템 → "Windows 자동시작 등록" 클릭
```

또는 명령 프롬프트에서:

```bash
python -m domae_mcp --install-startup
```

### 해제

```bash
python -m domae_mcp --uninstall-startup
```

> 자동 실행 등록 후에는 PC를 켜면 트레이에서 조용히 실행됩니다.
> 브라우저에서 http://localhost:5900 접속하면 됩니다.

## 11. 업데이트 방법

새 버전이 나오면 프로그램 상단에 알림이 표시됩니다.

### 간편 설치로 설치한 경우

1. 기존 폴더 삭제 (설정은 유지됨)
2. 새 버전 다운로드 → 압축 해제
3. `install.bat` 실행
4. `start.bat` 실행

> 설정(도매 계정, 텔레그램 등)은 `~/.maipharm-domae-mcp/`에 저장되므로
> 프로그램을 다시 설치해도 설정이 유지됩니다.

### 수동 설치로 설치한 경우

```bash
cd maipharm-domae-mcp
git pull
pip install -e .
```

## 12. 문제 해결

### "포트가 이미 사용 중입니다"

다른 프로그램이 5900 포트를 쓰고 있는 경우:

```bash
python -m domae_mcp --port 5901
```

### "도매 로그인 실패"

1. 해당 도매 웹사이트에서 직접 로그인이 되는지 확인
2. 비밀번호를 최근에 변경했다면 설정에서 업데이트
3. [설정] → [테스트] 버튼으로 재확인

### "검색 결과가 없습니다"

1. 도매 계정이 등록되어 있는지 확인 (설정 → 도매 계정)
2. 검색어를 다르게 시도 (보험코드, 제품명 일부)
3. 해당 도매에 실제로 재고가 있는지 도매 사이트에서 직접 확인

### "프로그램이 실행되지 않습니다"

1. Python이 설치되어 있는지 확인:

```bash
python --version
```

2. Python 3.10 이상이어야 합니다
3. 프로그램을 다시 설치해보세요

### 기타 문제

- GitHub Issues에 문의: https://github.com/maipharm/maipharm-domae-mcp/issues
- 이메일: kjh@maipharm.com

## 13. 클라우드 서비스 (유료)

로컬 버전의 제한사항:
- PC가 켜져있을 때만 동작
- 모니터링 간격 최소 60분
- 텔레그램만 지원

**클라우드 서비스**를 이용하면:

| | Free (무료) | Basic (구독자 무료) | Pro (월 5만원) |
|---|:---:|:---:|:---:|
| 설치 필요 | O | X | X |
| 운영 시간 | PC 켜져있을 때 | **24시간** | **24시간** |
| 모니터링 간격 | 60분 | **30분** | **5분** |
| 알림 | 텔레그램 | 텔레그램 | 텔레그램 + **카카오톡** |
| 검색 이력 | 로컬 7일 | 90일 | 1년 |

### 가입 방법

1. 팜스퀘어(pharmsquare.com) 유료회원 가입
2. 메뉴에서 "도매 통합검색" 클릭
3. 도매 계정 등록 후 바로 사용

> PC 설치 없이, 어디서든 브라우저로 접속하여 사용할 수 있습니다.
